# HR Analytics AI Agent — Database Final Project

## Team
Bekibaeva Aigerim | Kunduzbekova Perizat | Aidai Totoeva

## Description
AI-powered HR Analytics system built on IBM HR dataset.
Analyzes employee attrition across 1,470 employees using MySQL + Python + Telegram Bot.

## Tech Stack
- MySQL 9.6 — relational database (5 tables, normalized)
- Python 3.13 + SQLAlchemy — database connection
- Jupyter Notebook — analysis environment
- Telegram Bot API — AI agent interface

## Key Insights
1. Sales department: highest attrition at 16.8%
2. Overtime workers leave 2.2% more often
3. 38.8% of employees stagnant 5+ years
4. Single 18-25 far commuters: 22% attrition risk

## How to Run
1. Run hr_analytics_full_setup.sql in MySQL Workbench
2. Open hr_ai_agent.py in Jupyter Notebook
3. Set your MySQL password and Telegram token
4. Run all cells
5. Open @hr_aiAgentBot in Telegram
